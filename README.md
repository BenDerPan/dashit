# DashIT，旨在构建一个通用的，科幻的展示平台，支持RESTFul+GraphQL等数据展示接口。

![./docs/img/overview.jpg](./docs/img/overview.jpg)


## 快速体验
* 安装依赖包： `yarn install`
* 开启调试预览：`yarn start`
* 访问：[ http://localhost:3000](http://localhost:3000/) 查看效果
